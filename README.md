# 사하구 · 부산광역시

> 사하구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-saha.github.io/**

사하구는 부산의 서남단, 낙동강이 남해로 흘러드는 하구에 자리한 자치구로 강과 바다가 한데 만나는 물의 고장입니다. 구 이름은 조선시대 동래부 사천면(沙川面)의 하단부(下端)에 해당한다 하여 사천의 '사(沙)'와 아래를 뜻하는 '하(下)'를 합쳐 붙여졌습니다. 낙동강 삼각주에 형성된 철새 도래지 을숙도, 백사장과 낙조로 이름난 다대포해수욕장, 계단식 산동네가 알록달록 물든 감천문화마을이 모두 이곳에 모여 있어 생태와 예술, 항만이 공존합니다. 서부산의 관문으로서 신평·장림 산업단지와 감천항을 품어 부산 경제의 한 축을 담당하기도 합니다. 자연의 원형과 도시의 활력이 나란히 살아 숨 쉬는 지역이 바로 사하구입니다.
